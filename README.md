  Library Management System (Java OOP Project)

 Project Overview
This is a console-based Library Management System developed using Java and Object-Oriented Programming (OOP) principles.

The system allows users to manage books and members efficiently through a menu-driven interface.

 Features

- Add Book
- Add Member
- Borrow Book
- Return Book
- Search Book by Title
- Search Book by Author
- Display All Books
- Display Available Books
- Book Availability Validation

---
Technologies Used

- Java
- OOP Concepts
- Collections Framework (ArrayList)
- Git & GitHub

---
 OOP Concepts Implemented

- Encapsulation (Private variables + getters/setters)
- Abstraction
- Object Composition
- Separation of Concerns
